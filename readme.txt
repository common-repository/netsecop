Netsecop is a cloud-based web application security scanner which scans your website against vulnerabilities and malware.


== Description ==

Netsecop Wordpress security plugin is the one-click solution to create your Netsecop.com account and start your security scans.

Netsecop Wordpress plugin offers you the following features at no charge:

* Web application vulnerability scanning
* Remote malware scanning
* Blacklist scanning
* Instant notifications

This plugin shows the security summary of your website on the WordPress dashboard. In order to get more detailed information about your security problems, you can login Netsecop.com with your WordPress email address. You will get an email regarding to your Netsecop.com account once you activate the plugin.

IMPORTANT: Netsecop Wordpress plugin relies on Netsecop.com servers in order to perform security scans and parse scan results. All related information will be securely transferred back and forth between Netsecop servers and the plugin. By activating the Netsecop Wordpress Plugin, you agree to our [terms and conditions](https://www.netsecop.com/termsandconditions.html).

== Installation ==

Installation of the Netsecop Wordpress plugin is pretty straightforward:

1. Log into your WordPress admin panel
2. Choose Plugins -> Add New
3. Search for "Netsecop" via the search bar
4. Click on "Install Now" button
1. Once the installation is completed, click on "Activate" button to automatically create your Netsecop.com account and start your scans.


== Frequently Asked Questions ==

= Why do I need to prove my website ownership? =

Netsecop discovers vulnerabilities in your website. In order to start vulnerability scan and access sensitive results, we need you to verify the ownership of the website. For this purpose we created a very tiny, semi-transparent Netsecop seal. Netsecop scanner will check the presence of the Netsecop seal before starting the vulnerability scan.

= What is the scope of vulnerability scan? =

Netsecop performs a very detailed web application vulnerability scanning which covers all the steps listed in OWASP TOP 10 list.

= What is remote malware scanning? =

Netsecop visits your web page and scans your web pages for injected malicious codes, harmful scripts, backdoors and exploit kits.

= What is blacklist scanning? =

Netsecop searches your website in various blacklist and spam databases.

= What are instant notifications? =

Netsecop sends you e-mail notifications about your scan progress and results.

= What about hidden fees? =

Netsecop provides you the features listed above for completely free. There is no hidden fees or extra charges.

= Will Netsecop decrease my site performance? Any other installations? =

Netsecop works in the cloud. It is extremely lightweight and doesn't slow down your website. There is no extra installation on your premises.
